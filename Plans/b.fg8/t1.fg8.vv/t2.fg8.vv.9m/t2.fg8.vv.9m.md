## Context

This Epic is additive — the new stdio MCP server, `spawn_worker`, and `list_spawned_workers` land alongside the existing daemon (which Epic `t1.fg8.bp` removes). One regression gate at the Epic level is the right shape: the three Tasks share the import surface (`waggle.mcp_stdio`, `waggle.engine`, `waggle.claude_status`), so per-Task gates would each run the full suite and surface the same pre-existing failures three times. A single end-of-Epic gate gives a clean diff.

## What Needs to Change

No new files. Run `pytest -q` from the repo root after all three Tasks of this Epic land. Triage any failures or collection errors and apply minimal fixes scoped to the new files only.

## Constraints (strict)

- **No modifications of any kind to pre-existing files** — no edits, no deletions, no skips, no xfails, no monkeypatch helpers, no `# type: ignore` additions. Pre-existing-test cleanup belongs to sibling Epics `t1.fg8.bp` and `t1.fg8.2g`.
- Acceptable fixes (all in new files only): narrowing imports in `mcp_stdio.py` so the module has no import-time side effects; deferring side-effectful calls; fixing test fixture cleanup; correcting interface mismatches in new test files.
- If any regression cannot be reconciled by editing only new files, agent reports the failure verbatim and declares this Subtask blocked. Agent does NOT mark the Subtask complete and does NOT mutate any pre-existing file.

## Acceptance Criteria

- `pytest -q` exits 0 after all three Tasks land.
- Test count ≥ pre-Epic count + new tests from Tasks 1, 2, 3; no pre-existing test deleted/skipped/xfailed.
- `python -c 'import waggle.mcp_stdio'` (or whatever path Engineer chose) succeeds with `subprocess.run`, `socket.socket`, and daemon-era modules patched to raise on call — proving zero subprocess fork, zero socket bind, zero daemon import at module load.
- The SR-2.2 startup capability check fires at `waggle mcp` subcommand entry, NOT at module import time (verifiable by the assertion above succeeding).
- No file outside this Epic's new files has been modified.
