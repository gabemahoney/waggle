## Context

The loader module added by Task 2's engineer subtask must be unit-tested. SR-11.6 mandates a new test file `tests/test_engine_templates_load.py` covering the loader's public surface and the SR-6.4 validation rules. The new helper `fake_templates_dir` and the `TEMPLATE_TOML_*` constants from the prior subtask are the inputs to these tests.

## What Needs to Change

Create `tests/test_engine_templates_load.py`. The file MUST use `fake_templates_dir` and the `TEMPLATE_TOML_*` constants ‚Äî no test in this file may hardcode a TOML string literal. Cover:

1. **Missing directory** ‚Äî call `fake_templates_dir({})` (or arrange for `_templates_dir()` to point at a path that does not exist). Assert: enumerate-all yields zero entries with no error; single-template lookup for any name returns `ErrTemplateNotFound` with description naming both the name and the searched directory.
2. **Valid load ‚Äî minimal** ‚Äî `fake_templates_dir({"orch": TEMPLATE_TOML_MINIMAL})`. Assert: single-template lookup for `"orch"` returns the parsed option map with the expected fields; enumerate-all yields one valid entry for `orch`.
3. **Valid load ‚Äî full** ‚Äî `fake_templates_dir({"full": TEMPLATE_TOML_FULL})`. Assert: the parsed option map contains every SR-1.1 option (excluding `template`) with the correct types (scalars as strings, maps as dicts, lists as lists).
4. **Malformed ‚Äî parse failure** ‚Äî `fake_templates_dir({"bad": TEMPLATE_TOML_MALFORMED_PARSE})`. Assert: single-template lookup for `"bad"` returns `ErrTemplateMalformed`; enumerate-all yields one error entry for `bad` (and continues iterating if other entries are present).
5. **Malformed ‚Äî unknown key** ‚Äî `fake_templates_dir({"bad": TEMPLATE_TOML_MALFORMED_UNKNOWN_KEY})`. Assert: `ErrTemplateMalformed` with description naming the unknown key.
6. **Malformed ‚Äî recursive `template` key** ‚Äî `fake_templates_dir({"bad": TEMPLATE_TOML_RECURSIVE})`. Assert: `ErrTemplateMalformed` (per SR-2.4 + SR-6.4 bullet 3).
7. **Malformed ‚Äî invalid `thinking` value** ‚Äî programmatically construct via `tomli_w` a body with `thinking = "huge"`. Assert: `ErrTemplateMalformed`; description names the four valid values (`low`, `medium`, `high`, `xhigh`).
8. **Malformed ‚Äî scalar type mismatch** ‚Äî programmatically construct a body with `cwd = 123` (int, not string). Assert: `ErrTemplateMalformed`.
9. **`_read_template_file` is a patchable seam** ‚Äî patch `_read_template_file` directly via `unittest.mock.patch` and confirm the loader delegates to it. Helps Task 3 tests rely on the seam.
10. **Enumerate-all mixed result** ‚Äî `fake_templates_dir({"good": TEMPLATE_TOML_MINIMAL, "bad": TEMPLATE_TOML_MALFORMED_PARSE})`. Assert: enumerate-all yields one valid entry for `good` and one error entry for `bad`; the malformed file does not abort iteration.
11. **Import-time no filesystem reads** ‚Äî assert that importing `claude_spawn.templates` does not call either `_templates_dir()` or `_read_template_file()`. Verify via mock spies installed BEFORE the import (use `importlib.reload` or `pytest.importorskip` patterns).

Every test in this file uses `fake_templates_dir`, `TEMPLATE_TOML_*` constants, or programmatic `tomli_w` construction ‚Äî NEVER a hardcoded TOML string literal. No test touches `~/.claude-spawn/templates/`.
