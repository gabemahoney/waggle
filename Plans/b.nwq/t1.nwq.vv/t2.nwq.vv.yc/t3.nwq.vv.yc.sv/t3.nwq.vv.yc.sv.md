## Context

SR-11.6 mandates a new test file `tests/test_engine_templates_write.py` exercising `write_template_impl` directly (impl-level coverage). This is separate from the MCP-level test file (sibling subtask). Every test uses `fake_templates_dir`, `TEMPLATE_TOML_*` constants, or programmatic `tomli_w` construction — NEVER hardcoded TOML literals. No test touches `~/.claude-spawn/templates/`.

## What Needs to Change

Create `tests/test_engine_templates_write.py`. Cover:

1. **Minimal success + round-trip:** call `write_template_impl("orch", {"cwd": "/tmp"}, force=False)` inside `fake_templates_dir({})`. Assert success dict has `path` and `options`; file exists; loading it via the Epic 3 loader returns `{"cwd": "/tmp"}`.
2. **Full-options round-trip:** pass an `options` dict exercising every supported SR-1.1 key (use `TEMPLATE_TOML_FULL` parsed via `tomllib` as the source). Write, then load via the loader; assert loaded options equal input options.
3. **Collision without `force`:** two consecutive calls with same name. Second returns `ErrTemplateExists`; description names the existing path; file content unchanged from first write.
4. **Collision with `force=True`:** second call with `force=True` overwrites. Assert new content reflects new options. Verify atomicity by injecting an error in the rename step (monkeypatch `os.rename` to raise) and asserting the canonical file still holds the original content.
5. **Unsafe-name shapes (five tests, each → `ErrTemplateNameUnsafe`, no file written):**
   - `name="foo/bar"` (path separator).
   - `name=".hidden"` (leading dot).
   - `name=".."` (equals `..`).
   - `name="foo/../escape"` (`..` substring).
   - `name=""` (empty).
6. **Unknown option key:** `options={"bogus": 1}` → `ErrTemplateOptionsInvalid`; description enumerates `bogus`; no file written.
7. **Recursive `template` key:** `options={"template": "x"}` → `ErrTemplateOptionsInvalid` (SR-2.4 — templates cannot reference templates); no file written.
8. **Invalid `thinking`:** `options={"thinking": "huge"}` → `ErrTemplateOptionsInvalid`; description names the four valid values.
9. **Type mismatch:** `options={"cwd": 123}` (int instead of string) → `ErrTemplateOptionsInvalid`.
10. **Directory auto-creation:** configure `fake_templates_dir` so the templates directory does not exist when `write_template_impl` is called (remove the temp dir after the seam is patched). Assert the function creates the directory and writes the file successfully.

Every test in this file MUST use `fake_templates_dir` (no real filesystem access) and either `TEMPLATE_TOML_*` constants or programmatic `tomli_w` construction (no hardcoded TOML literals).
