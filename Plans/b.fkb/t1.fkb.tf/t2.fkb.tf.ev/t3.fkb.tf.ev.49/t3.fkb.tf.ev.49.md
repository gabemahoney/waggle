## Context
The Python package directory currently lives at `src/waggle/`. Every cross-module import in the source tree references it as `waggle.X`. The Epic atomically moves the directory to `src/claude_spawn/` and flips every source-side import. This subtask is the foundational move; B/C/D/E all depend on the new package path existing.

The directory contains seven module files post-Epic 1: `__init__.py`, `cli.py`, `mcp_stdio.py`, `spawn.py`, `claude_status.py`, `sting.py`, `installer.py`.

## What Needs to Change
- Move the package directory using `git mv src/waggle src/claude_spawn` so blame and history follow the rename. Do NOT `rm -rf` + `cp -R` — that loses git history.
- In every module under `src/claude_spawn/`, update cross-module imports from `from waggle.X import Y` / `import waggle.X` / `import waggle` to `claude_spawn.*` form. The deferred-import sites in `cli.py` (`from waggle import sting`, `from waggle import mcp_stdio`, `from waggle import installer`) flip too.
- Update dotted-path strings inside source-side docstrings that name patch seams (e.g. prose like "tmux invocations go through `waggle.spawn._tmux`") so they reference `claude_spawn.*`. These are documentation strings, not data values.
- Do NOT touch `pyproject.toml` (Subtask B), the argparse parser class or CLI help text (Subtask C), prose mentions of "Waggle" inside docstrings/log/error messages (Subtask D), test files (Subtask E), or any of the deferred data strings listed in the parent Task.

## Key Files
- `src/waggle/` → `src/claude_spawn/` — directory move via `git mv`
- `src/claude_spawn/cli.py` — flip deferred-import paths
- `src/claude_spawn/mcp_stdio.py` — flip `from waggle import spawn` and any dotted-path docstring references
- `src/claude_spawn/spawn.py` — flip `from waggle import claude_status` and any dotted-path docstring references
- `src/claude_spawn/sting.py` — flip `from waggle import claude_status` and any dotted-path docstring references
- `src/claude_spawn/claude_status.py` — flip any dotted-path docstring references
- `src/claude_spawn/installer.py` — flip any dotted-path docstring references

## Acceptance Criteria
- `ls src/waggle 2>&1 | grep -c "No such"` returns 1; `ls src/claude_spawn/` lists all seven module files.
- `git log --follow src/claude_spawn/cli.py` shows commits from before the rename (history preserved via `git mv`).
- `grep -RIn "from waggle\b\|import waggle\b" src/` returns zero matches.
- `python -c "import claude_spawn"` succeeds; `python -c "import claude_spawn.cli"` succeeds.
- `python -c "import waggle"` raises `ModuleNotFoundError` (the old name no longer resolves).
