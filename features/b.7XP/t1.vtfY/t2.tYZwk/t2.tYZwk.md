## Context
The send_command MCP tool needs to send keystrokes to tmux panes. The tmux.py module must be extended with send functions. These build on the pane capture and validation functions added in Epic t1.kN24.

## What Needs to Change
- Add send_keys function to `src/waggle/tmux.py`: send text to a pane with optional enter keystroke using `pane.send_keys(text, enter=True)`
- Add input clearing function: send Ctrl+C to clear any partial input before sending new text (SR-5.3)
- Accept pane_id parameter for multi-pane targeting (reuse pane resolution from t1.kN24)
- All async-safe via `asyncio.to_thread()` following existing module pattern
- Error handling: catch libtmux exceptions, return structured errors
- Add tests to test_tmux.py (file already exists from t1.3dyn)

## Success Criteria
- send_keys delivers text + Enter to mocked pane
- Input clearing sends Ctrl+C correctly
- Pane targeting works (specific pane_id or active pane)
- Errors caught and returned as structured dicts
- Tests pass
