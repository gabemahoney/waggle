## Context
This module replaces all raw subprocess tmux calls. It provides async-safe wrappers for libtmux operations. The module is separate from `server.py` per SR-1.3. Downstream Epics (read_pane, send_command, spawn_agent, close_session) will extend this module with additional functions.

## What Needs to Change
- Create `src/waggle/tmux.py` with functions for:
  - **Session enumeration with paths**: Return list of dicts with session_name, session_id, session_created, session_path — matching the data previously extracted from `TMUX_FMT_WITH_PATH` subprocess output. This is used by `list_agents` for enrichment.
  - **Active session composite keys**: Return set of `{name}+{session_id}+{created}` strings — matching the data previously extracted from `TMUX_FMT_KEYS_ONLY` subprocess output. This is used by `cleanup_dead_sessions` for orphan detection.
  - **LLM detection**: Accept a pane object and return bool — True if `pane_current_command` equals `claude` or `opencode` (case-insensitive). Per SR-8.1: single tmux query only, no process tree walking, no pgrep/psutil.
  - All functions wrapped with `asyncio.to_thread()` for async safety. Multiple sequential libtmux ops bundled into single sync functions before offloading (SR-1.2).
  - Error handling: catch `LibTmuxException` subclasses and generic exceptions from `QueryList.get()` — known issue where some exceptions don't inherit from `LibTmuxException` (libtmux issue #541). Convert to structured error responses (SR-10.1).
  - Graceful tmux-unavailable: if `libtmux.Server()` fails because no tmux server is running, return empty results, not crash (SR-10.2).

## Acceptance Criteria
- Module imports cleanly with no side effects
- Session enumeration returns correct data structure when tmux is available
- Returns empty results gracefully when tmux is unavailable
- LLM detection returns True for `claude`/`opencode` (including case variations like `Claude`), False for `zsh`/`bash`/other
- All exceptions are caught and converted to structured responses — no raw libtmux exceptions propagate
